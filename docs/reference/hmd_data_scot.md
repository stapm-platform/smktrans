# Death rates for Scotland

Data from the Human Mortality Database on the 1x1 age and period central
death rates (mx) for the Scotland population. Death rates are stratified
by sex only. Ages 16-89 selected

## Usage

``` r
hmd_data_scot
```

## Format

A data table.

## Source

[Human Mortality Database -
Scotland](https://www.mortality.org/cgi-bin/hmd/country.php?cntr=GBR_SCO&level=1)
