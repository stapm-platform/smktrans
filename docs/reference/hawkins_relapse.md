# Long-term probabilities of smoking relapse

Probabilities of smoking relapse estimated from Hawkins 2010, who
analysed the British Household Panel Survey.

## Usage

``` r
hawkins_relapse
```

## Format

A data table.

## Source

Hawkins J, Hollingworth W, Campbell R (2010). “Long-term smoking
relapse: a study using the british household panel survey.” *Nicotine &
tobacco research*, **12**(12), 1228–1235.
