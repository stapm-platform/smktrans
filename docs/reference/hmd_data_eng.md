# Death rates for England & Wales

Data from the Human Mortality Database on the 1x1 age and period central
death rates (mx) for the England & Wales civilian population. Death
rates are stratified by sex only. Ages 11-89 selected

## Usage

``` r
hmd_data_eng
```

## Format

A data table.

## Source

[Human Mortality Database - England and
Wales](https://www.mortality.org/cgi-bin/hmd/country.php?cntr=GBRCENW&level=1)
